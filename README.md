# Генератор безопасных паролей

Простой и надежный инструмент для создания криптографически стойких паролей на Python.

## Для чего это нужно
- **Безопасность**: Генерирует сложные пароли, которые трудно взломать.
- **Удобство**: Мгновенное создание пароля любой длины.
- **Гарантия состава**: Каждый пароль содержит буквы (верхний и нижний регистр), цифры и спецсимволы.

## Установка
1. Убедитесь, что у вас установлен Python 3.
2. Скачайте файл `password_generator.py`.

## Использование
Запустите скрипт в терминале:
```bash
python password_generator.py
```

## Как это работает
Скрипт использует модуль `random` для выбора случайных символов.
